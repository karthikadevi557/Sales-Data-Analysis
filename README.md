# Sales Data Analysis

##  Overview

This project analyzes 10,000+ sales records using Python to extract meaningful business insights.

##  Tools Used

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Jupyter Notebook

##  Key Analysis

* Revenue calculation
* Top-performing products
* Regional sales analysis
* Sales trend over time
* Moving average & peak detection

##  Visualizations

Includes:

* Sales trend graph
* Monthly analysis
* Region comparison
* Peak sales highlight

##  Conclusion

This project demonstrates practical data analysis, visualization, and business insight generation.
